Capacidade Técnica — DEV

✓ Positivos ✗ Problemas → Solução proposta
Formulário de chamado é tecnicamente simples e pode ser implementado com as tecnologias que o time já conhece.
Integração com a grade horária real do campus é complexa e depende de dados externos da instituição.
Versão inicial com inserção manual das salas e horários pela coordenação; integração com grade fica para versões futuras.
Calendário de disponibilidade é essencialmente um CRUD básico com visualização, acessível para o time.
Atualização em tempo real da disponibilidade das salas exige infraestrutura adicional.
Implementar real-time em versões futuras. Na versão inicial, usar polling simples com atualização periódica.
A equipe pode convidar colaboradores externos, colegas, professores e monitores, para apoiar o desenvolvimento.
A equipe tem pouca experiência com deploy e infraestrutura de aplicações web em ambiente de produção.
Buscar apoio de professores ou Centro Acadêmico para viabilizar o deploy institucional e a manutenção do servidor.
